《原神》模拟抽卡机器人。

基于 [YuQ](https://github.com/YuQWorks/YuQ) 开发。

指令：
```
可莉十连
可莉单抽

狼末十连
狼末单抽

标配十连
标配单抽
```

食用方法：
```
clone。
编辑 ./src/main/resource/conf/bot.properties。
填写 QQ 账号与 QQ 密码。
./gradlew build
java -jar ./build/libs/YuanShenBot-1.0-SNAPSHOT-all.jar
```
